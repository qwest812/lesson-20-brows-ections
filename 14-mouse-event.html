mousedown Кнопка мыши нажата над элементом. mouseup Кнопка мыши отпущена над элементом. mouseover Мышь появилась над элементом. mouseout Мышь ушла с элемента. mousemove Каждое движение мыши над элементом генерирует это событие. click Вызывается при клике мышью, то есть при mousedown, а затем mouseup на одном элементе contextmenu Вызывается при клике правой кнопкой мыши на элементе. dblclick Вызывается при двойном клике по элементу. Порядок срабатывания событий mousedown → mouseup → click. event.which == 1 – левая кнопка event.which == 2 – средняя кнопка event.which == 3 – правая кнопка
Правый клик на этой кнопке выведет "Клик".
